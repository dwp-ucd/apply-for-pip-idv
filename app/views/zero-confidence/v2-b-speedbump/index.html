Design iteration for zero confidence V2 - Part B (with speedbump)

Prototype

This version contains a speedbump page, to let users know they will need to register again.

We tested the user journey for scenario 4b. We asked the same participants from Part A to return to their PIP application partway through.

We tested users returning with PIP creds being asked to register for DTH creds before signing in to continue their application.
